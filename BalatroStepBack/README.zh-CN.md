# Balatro Step Back / 对局回退

[English README](./README.md)

作者：ZhiSunian

版本：1.2.1

## 功能

Balatro Step Back / 对局回退会在当前盲注内添加检查点，用来回到出牌、弃牌或使用手持消耗牌之前。

- 英文 UI：`Back` 回到最近一次检查点，`History` 打开检查点列表。
- 简体中文 UI：`回退` 回到最近一次检查点，`记录` 打开检查点列表。
- 繁体中文 UI：`回退` 回到最近一次检查点，`記錄` 打开检查点列表。
- 历史列表会显示为“回到第 1 次出牌前”“回到第 1 次使用消耗牌前”等直观格式。
- “记录”菜单会分页显示检查点，默认折叠；点击“详情”后才展开具体牌面，避免记录过多时挤出屏幕。
- 出牌和弃牌检查点在“详情”里会显示缩小后的真实 Balatro 牌面；出牌检查点还会显示识别到的牌型。
- 历史记录仅限当前对局的当前盲注。选择更早的检查点后，之后的检查点历史会被丢弃。
- 当前盲注的检查点历史会随对局一起保存，因此回到主界面后继续同一局，仍可看到并回退到之前的检查点。
- 在盲注内使用手持消耗牌前也会创建检查点，因此回退可以把塔罗牌、星球牌、幻灵牌等恢复回消耗牌区。

## 依赖

- PC 版 Balatro
- Lovely
- Steamodded / SMODS

本压缩包不包含 Balatro、Lovely、Steamodded 或任何游戏文件。

## 安装

1. 安装 Lovely。
2. 安装 Steamodded / SMODS。
3. 解压本压缩包。
4. 将 `BalatroStepBack` 文件夹放入：

```text
%AppData%\Balatro\Mods
```

最终结构应为：

```text
%AppData%\Balatro\Mods\BalatroStepBack\manifest.json
%AppData%\Balatro\Mods\BalatroStepBack\main.lua
%AppData%\Balatro\Mods\BalatroStepBack\README.md
%AppData%\Balatro\Mods\BalatroStepBack\README.zh-CN.md
```

5. 启动 Balatro，并确认 Steamodded 模组列表中出现 `Balatro Step Back` 或 `Step Back / 对局回退`。

## 兼容性说明

本模组包装 Balatro 原有的出牌、弃牌和用牌函数，并复用 Balatro 的对局保存/读取数据结构。它不会修改 Balatro、Steamodded 或 Lovely 的安装文件。

已知限制：

- 历史记录仅限当前对局的当前盲注。
- 恢复检查点时会短暂重新载入对局状态。
- 保存额外外部状态的其他模组可能无法被完全回退。
- 如果发生崩溃，请先只启用 Lovely、Steamodded 和本模组复测。

## 发布说明

本压缩包只包含原创 Lua 模组代码和元数据，不包含 Balatro、Lovely、Steamodded、游戏文件、图片、音频或其他第三方资产。
