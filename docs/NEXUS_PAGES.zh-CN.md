# Nexus Mods 页面草稿

[English version](./NEXUS_PAGES.md)

这些草稿可以直接复制到 Nexus Mods 页面中。每个简要介绍和完整描述都是中英双语，并且完整描述采用“英文完整一遍，中文完整一遍”的结构。

常用链接：

- Lovely: https://github.com/ethangreen-dev/lovely-injector
- Steamodded / SMODS: https://github.com/Steamodded/smods
- Steamodded 安装指南: https://www.nexusmods.com/balatro/articles/31
- GitHub 仓库: https://github.com/kekeasure/balatro-custom-mods

## 可复制文本

具体可复制文本请使用 [英文版草稿](./NEXUS_PAGES.md)。其中已经包含两个模组的中英双语简要介绍和完整描述，并说明英文、简体中文、繁体中文 UI，适合直接粘贴进 Nexus 的输入框。

## 文件名

- Balatro Step Back / 对局回退: `BalatroStepBack-1.0.0.zip`
- Balatro Score Preview / 分数预览: `BalatroScorePreview-1.2.0.zip`

## 推荐分类和标签

- Balatro Step Back / 对局回退: `Gameplay` 或 `User Interface`；标签建议 `Quality of Life`、`Gameplay`、`User Interface`。
- Balatro Score Preview / 分数预览: `User Interface` 或 `Gameplay`；标签建议 `Quality of Life`、`Gameplay`、`User Interface`。
