# Balatro Modifier Warning / 覆盖提醒

作者：ZhiSunian

中文名：覆盖提醒

版本：1.1.2

当当前选中的消耗牌会把目标扑克牌已有的同槽位效果覆盖掉时，在目标牌附近显示按语言适配的覆盖警告标签。标签会直接写出“覆盖提醒”以及“旧效果 → 新效果”，例如“黄金牌 → 玻璃牌”。

## 会提示的情况

- 增强牌被替换，例如对已经是钢铁牌、黄金牌、幸运牌、玻璃牌、万能牌、奖励牌、倍率牌或石头牌的牌使用另一张增强塔罗牌。
- 蜡封被替换，例如对已经有其它蜡封的牌使用 Talisman、Deja Vu、Trance、Medium。

这个模组不会阻止操作，也不会修改游戏规则，只在被选中的目标牌附近加视觉提示。

## 不会提示的情况

- The Star、The Moon、The Sun、The World 这类改花色的塔罗牌。
- 改点数、复制牌、摧毁牌、小丑牌版本效果或其它危险操作。
- 背面牌或隐藏牌，避免泄露玩家本不该知道的信息。

## 安装

1. 安装 Lovely 和 Steamodded。
2. 将 `BalatroModifierWarning` 文件夹复制到 Balatro 的 `Mods` 文件夹。
3. 重启游戏。

## 兼容性

本模组使用 Steamodded 的绘制步骤，只读取当前牌状态。大多数模组应可兼容。使用标准 `mod_conv` 字段转换增强牌的自定义消耗牌会自动被覆盖提示识别；完全自定义脚本效果后续可能需要单独适配。

本压缩包只包含原创 Lua 模组代码和元数据，不包含 Balatro、Lovely、Steamodded、游戏文件、游戏素材、音频、存档或其它第三方资产。
